# **题库录制格式**
本题库中的题目采用统一的格式以与本[**诗词大会程序**](../Conference.htm)兼容。不同题目类型有不同的题库录制格式。
## **choose 选择题**
```js    
{
    type: choose /* 表明这是选择题 */,
    note: "“停车坐爱枫林晚”中，“坐”的意思是？" /* 这是题目的注释部分 */,
    option: {
        A: "坐下" /* A选项，必选 */,
        B: "座位" /* B选项，必选 */,
        C: "因为" /* C选项，必选 */,
        D: "无含义" /* D选项，可选 */,
    },
    blankChar: "" /* 乱序重组字符串，无用处，不用干涉 */,
    answer: C /* 答案 */,
    from: "杜牧《山行》" /* 诗词出处。可选。
    建议：应使用书名号《》包裹内容，以方便程序添加通向百度汉语中对应诗词的连接 */,
    used: false /* 使用情况，不用干涉 */
},
```

## **space 填空题**
```js    
{
    type: space /* 表明这是填空题 */,
    note: "《约客》是space写的。“黄梅时节家家雨，space”体现乡村之景是清新恬静、和谐美妙的。" 
    /* 这是题目的注释及填空部分。使用space字样代替答题框。
    注意：如果存在多个答题框，下方答案顺序必须与答题框顺序一致！ */,
    option: {
        A: "" /* 选择题选项，无用处，不用干涉 */,
        B: "" /* 选择题选项，无用处，不用干涉 */,
        C: "" /* 选择题选项，无用处，不用干涉 */,
        D: "" /* 选择题选项，无用处，不用干涉 */,
    },
    blankChar: "" /* 乱序重组字符串，无用处，不用干涉 */,
    answer: "赵师秀 青草池塘处处蛙" 
    /* 答案。如果存在多个答题框，使用空格 分割每个答题框对应的答案。
    注意：上方答题框顺序必须与答案顺序一致！
    如果在题目中已经添加诗词的句读号，请注意不要在答案中重复添加！ */,
    from: "赵师秀《约客》" /* 诗词出处。可选。
    建议：应使用书名号《》包裹内容，以方便程序添加通向百度汉语中对应诗词的连接 */,
    used: false /* 使用情况，不用干涉 */
},
```

## **blank 乱序重组题**
```js
{
    type: blank /* 表明这是乱序重组题 */,
    note: "" 
    /* 注释及填空部分，无用处，不用干涉 */,
    option: {
        A: "" /* 选择题选项，无用处，不用干涉 */,
        B: "" /* 选择题选项，无用处，不用干涉 */,
        C: "" /* 选择题选项，无用处，不用干涉 */,
        D: "" /* 选择题选项，无用处，不用干涉 */,
    },
    blankChar: "千一里广江安陵得间厦日" /* 乱序重组字符串。
    将乱序诗词单字依次填入此字符串中。中间不用加入任何间隔符号。
    注意：目前仅支持9字、12字、15字、16字格式乱序字符串。请注意题目长度！ */,
    answer: "安得广厦千万间"
    /* 答案。请按照原题目答案进行顺序填写。
    注意：由于乱序重组题目一般只要求识别出一句诗词，请不要在这中间添加句读！ */,
    from: "陆游《茅屋为秋风所破歌》" /* 诗词出处。可选。
    建议：应使用书名号《》包裹内容，以方便程序添加通向百度汉语中对应诗词的连接 */,
    used: false /* 使用情况，不用干涉 */
},
```
